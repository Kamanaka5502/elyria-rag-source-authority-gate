# Security Policy

## Public-Safe Repository

This repository is a public-safe reference architecture and sandbox for enterprise RAG source authority review.

Do not submit:

- credentials
- API keys
- access tokens
- customer data
- private implementation details
- protected Elyria Systems runtime machinery
- confidential diagrams
- production logs
- regulated data

## Production Use Boundary

This repository is not a plug-and-play production enforcement system. It is a reference implementation and pilot sandbox.

Production use requires organization-specific security review, identity design, logging design, privacy review, legal review, infrastructure controls, incident process, retrieval governance, and source authority mapping.
