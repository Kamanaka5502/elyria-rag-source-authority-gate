# Visual Index

## Elyria RAG Source Authority Gate

This index gives the repository a fast executive scan path.

---

## Control Signal

| Signal | Meaning |
|---|---|
| **Source Authority** | Retrieval is governed by approved source ownership and authority. |
| **Access Scope** | Retrieval respects identity, permission, and sensitivity boundaries. |
| **Freshness** | Source content is current enough for operational use. |
| **Grounding** | Output can be supported by approved source evidence. |
| **Telemetry** | Retrieval and output support can be reconstructed. |
| **Revalidation** | Changed source, corpus, model, prompt, permission, or policy triggers renewed review. |

## Decision Visuals

```text
ADMIT      Approved source, authorized access, current content, grounded output.
HOLD       Missing owner, freshness, grounding, telemetry, or review evidence.
REFUSE     Source not approved, access unauthorized, or sensitive controls missing.
REVALIDATE Source, corpus, permission, prompt, model, policy, or environment changed.
```
